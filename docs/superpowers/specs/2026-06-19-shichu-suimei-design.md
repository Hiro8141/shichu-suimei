# 四柱推命占いアプリ 設計書

- 作成日: 2026-06-19
- ステータス: 設計確定（実装計画待ち）
- プロジェクト仮名: shichu-suimei

## 1. 概要

生年月日と出生時間（任意）から四柱推命の**命式**を計算し、性格・才能・適職などを診断する一般向け無料PWA。GitHub Pagesで公開する静的サイト（サーバー・外部APIなし）。すべてブラウザ内で完結する。

世界観は「宇宙テーマ」。四柱推命の「星」「運命」のイメージに合わせ、夜空・星々・星座をモチーフにした神秘的で本格的なデザインとする。

### ターゲット / 方針（確定事項）

| 項目 | 決定 |
|---|---|
| 対象 | 一般向け無料PWA（GitHub Pages公開） |
| 占いの深さ | スタンダード：命式（四柱）＋通変星・十二運まで。性格・才能・適職を多面的に診断 |
| 出生時間が不明な場合 | 時柱を省略し3柱で診断。「時間が分かればもっと詳しく出せます」と案内 |
| 診断文 | 要素別の短文を組み合わせる構造。原稿は本プロジェクトで執筆（静的データとして保持） |
| 保存・シェア | ローカル保存（複数人）＋ 結果の画像シェア の両方 |
| 計算エンジン | `lunar-javascript`（実績ある暦ライブラリ）を採用 |
| 対応デバイス | iPhone SE（375px幅）〜 iPad（768〜1024px）。タブレット縦が基準 |
| デザイン | 宇宙テーマ（夜空・星・星座）。本格的で神秘的 |

## 2. 全体構成

- 技術：HTML / CSS / JavaScript の静的PWA（既存作品と同じ構成）
- 画面は2つ：
  1. **入力画面** … 名前（任意）、生年月日、出生時間（任意）を入力
  2. **結果画面** … 命式表＋診断文＋保存/シェアボタン
- 外部サーバー・API通信なし。`lunar-javascript` はバンドルまたはCDNから読み込み、計算はクライアント内で実行
- PWA化（manifest＋Service Worker）でオフライン利用・ホーム追加に対応

## 3. 命式計算エンジン

### 3.1 ライブラリによる干支算出（`lunar-javascript`）

入力（生年月日＋任意の時刻）から、年・月・日・時の各柱の**十干十二支**を算出する。

- 年柱は**立春**で切り替わる、月柱は各**節入り**で切り替わる、という四柱推命の暦ルールをライブラリが正しく処理する
- 日柱は60日周期の干支、時柱は日干＋時刻から決定
- 時刻が未入力の場合は**時柱を計算しない**（3柱で診断）

### 3.2 自作判定ロジック（ライブラリの上に実装）

ライブラリが返す十干十二支を入力に、以下を自前で判定する。

- **通変星**（10種：比肩・劫財・食神・傷官・偏財・正財・偏官・正官・偏印・印綬）
  - 日干と各柱の天干（および蔵干）との五行関係・陰陽から判定
- **十二運**（12種：長生・沐浴・冠帯・建禄・帝旺・衰・病・死・墓・絶・胎・養）
  - 日干と各柱の地支の組み合わせから、十二運表に基づき判定
- **蔵干**：各地支に内包される干（地支ごとの定義表を持つ）
- **五行バランス**：命式中の干支を木・火・土・金・水に集計し強弱を出す

判定ロジックは命式オブジェクトを入力に取り、純粋関数として実装する（テスト容易性のため）。

### 3.3 命式データモデル（概念）

```
Meishiki {
  pillars: { year, month, day, hour? }   // hour は時刻未入力時 undefined
  // 各 pillar: { stem(天干), branch(地支), hiddenStems(蔵干[]),
  //             tenStar(通変星), twelveStage(十二運) }
  dayMaster: stem                          // 日干（診断の中心）
  fiveElements: { wood, fire, earth, metal, water }  // 五行集計
  hasHourPillar: boolean
}
```

## 4. 診断文データ

### 4.1 方針

要素ごとの短文を JSON で持ち、命式に該当する短文を拾って組み立てる。原稿は四柱推命の解釈に沿って本プロジェクトで執筆する。リアルタイムAI生成は行わない（静的サイトのため）。

### 4.2 データ構造（概念）

- `dayMaster`（日干10種）… 基本性格のメイン文章（リッチめ）
- `tenStar`（通変星）… 才能・適職などの補足短文
- `fiveElements` … 五行の傾向に関する短文
- 必要に応じて `twelveStage`（十二運）の短文

結果画面では「日干の基本性格 → 通変星から才能/適職 → 五行傾向」の順で短文を結合して表示する。

## 5. 画面・UX

### 5.1 入力画面

- 名前（任意）
- 生年月日（必須）… 年・月・日の入力
- 出生時間（任意）… 「わからない」を選べるようにする
- 「占う」ボタンで結果画面へ

### 5.2 結果画面

- ヘッダー：生年月日（と時刻）、「あなたの日主は〇〇」のキャッチ
- **命式表**：列＝年柱・月柱・日柱・時柱、行＝天干・地支・通変星・十二運（時柱なしの場合は時柱列を出さないか「―」表示）
- **基本性格**：日干中心の診断文
- **才能 / 適職 / 五行**のサマリーカード
- **保存する** / **画像でシェア** ボタン

### 5.3 レスポンシブ

- 基準：iPad 縦（768px〜）。横幅を活かしてゆったり配置
- 下限：iPhone SE（375px幅）。命式表が崩れないよう、列幅・フォントを調整（`table-layout: fixed` 等）し横スクロールを避ける
- タップ操作前提（タブレットファースト）

### 5.4 デザイン（宇宙テーマ）

- 背景：深い夜空（濃紺〜紫のダーク基調）に星々を散りばめる
- 干支の漢字：金/銀で輝かせ、神秘的に見せる
- 五行：宇宙の色で表現（木=青緑、火=赤、土=黄、金=銀、水=青）を星のように
- 星座モチーフをアクセントに使う
- 全体トーン：これまでの明るい子ども向けとは異なる「本格的・神秘的」

## 6. 保存・シェア

- **保存**：`localStorage` に複数人分（名前＋生年月日＋時刻＋算出結果）を保存。一覧から呼び出して再表示できる
- **シェア**：結果カードを `canvas` で画像化し、ダウンロード／`Web Share API` でLINE等へ共有

## 7. テスト

- 計算エンジンの**単体テスト**：手計算済みの命式や既知の生年月日に対し、四柱・通変星・十二運・五行集計が正しいかを突き合わせる
- 時刻あり／なし両パターンを検証（時柱の有無）
- 立春・節入りの境界日付（年・月が切り替わる前後）を重点的に検証
- レスポンシブ：iPhone SE 幅と iPad 幅で命式表が崩れないことを確認
